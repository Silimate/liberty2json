# Liberty2JSON STA

A rewrite of [liberty2json](https://github.com/silimate/liberty2json) to use the
OpenSTA liberty parser for maximum speed and compatibility.

## Usage

```sh
./build/src/liberty2json --outfile <json out> <liberty in>
```
